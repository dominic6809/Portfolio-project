About Our Project

This project was inspired by the need for streamlined task management during a busy academic term. As a Portfolio Project for Holberton School, the aim was to create a tool that could handle everything from individual task management to team-based projects. Our goal was to eliminate the clutter and confusion caused by juggling multiple deadlines.

We started this project in June 2024, when we noticed that managing tasks for multiple assignments, alongside collaboration in group projects, was chaotic without a centralized platform. This inspired us to build TaskManager—your solution to a more organized and efficient workflow.
